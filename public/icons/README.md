# Icons

Please add the following icon files to this directory:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

These are required for the Chrome Extension.

You can use any icon generator tool or create them manually.
A simple approach is to use a text logo "📝" on a gradient background.

For now, the extension will still work but with missing icons.
