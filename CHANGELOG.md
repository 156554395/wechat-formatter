# 更新日志

## [1.0.0] - 2026-03-13

### 新增 ✨

#### 核心功能
- ✅ 基础富文本编辑器（基于 Tiptap）
- ✅ 标题样式支持（H1-H6）
- ✅ 代码高亮功能（支持多种语言）
  - JavaScript
  - TypeScript
  - Python
  - Java
  - Go
  - Rust
  - HTML
  - CSS
  - SQL
  - Bash
- ✅ 代码主题支持
  - GitHub
  - Monokai
  - Dracula
  - One Dark
  - Nord
- ✅ 图片排版功能
  - 图片对齐（左、中、右）
  - 图片样式（默认、阴影、边框、阴影+边框）
- ✅ 一键复制到微信编辑器

#### 技术栈
- ✅ Vue 3 + TypeScript
- ✅ Tiptap 编辑器
- ✅ Prism.js / Lowlight 代码高亮
- ✅ Element Plus UI 组件库
- ✅ Vite 构建工具
- ✅ Chrome Extension V3

#### 项目配置
- ✅ 完整的 TypeScript 配置

- ✅ 开发和生产构建脚本

#### 文档
